# Physical Intelligence Interview Questions

Company-specific practice folders for Physical Intelligence roles.

- `runtime_software_engineer/`: candidate-calibrated Pi runtime loop covering C++ native-handle lifetime, DMABUF zero-copy memory, NPU/system latency profiling, control-loop jitter, SPSC coding, and research/runtime negotiation.
- `controls_engineer/`: feedback control, model-based control, real-time loops, actuator/sensor bring-up, and sim-to-real validation.
- `runtime_architecture/`: C++ real-time architecture drills such as double-buffered trajectory handoff, lock-free logging, and servo-loop safety.
