# Personal Answers Workspace

Use this folder for your own responses without modifying official materials.

- Question source files are in `docs/practice/questions/`.
- Official hints are in `docs/practice/hints/`.
- Official answers are in `docs/practice/answers/`.
- Your personal copies live here with two sections:
  - `## My Answer`
  - `## My Comments`
- Each personal file also includes `## Scoring Rubric` with three scoring tracks:
  - Interviewer scoring (1-5)
  - Agentic-AI scoring (1-5)
  - Self scoring (1-5)

Recommended flow per question:
1. Open the personal file in this folder.
2. Write your answer under `## My Answer`.
3. Add review notes under `## My Comments`.
4. Compare with official answer only after you finish.
